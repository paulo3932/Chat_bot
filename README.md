# 🛍️ Loja C & IA - Assistente de Vendas

Um sistema completo de e-commerce desktop que simula uma loja de roupas com atendimento via chatbot integrado. Desenvolvido com Python e CustomTkinter, oferece uma experiência de compra fluida com catálogo interativo, carrinho dinâmico e emissão de notas fiscais.

## ✨ Funcionalidades

- **🖥️ Interface moderna** - Design escuro com tema roxo, desenvolvido com CustomTkinter
- **📦 Catálogo de produtos** - Busca por nome ou categoria em tempo real
- **🛒 Carrinho de compras** - Adicione, remova ou altere quantidades dinamicamente
- **🤖 Chatbot assistente** - Interaja com o assistente virtual que responde às suas ações
- **💰 Checkout completo** - Calcule frete por região (Fortaleza/CE) e escolha forma de pagamento
- **📄 Notas fiscais** - Geração automática de nota fiscal em JSON
- **📞 Suporte ao cliente** - Envie mensagens para o suporte que são salvas localmente
